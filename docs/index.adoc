== Кратко

Вход — отчёт института о хвостах (`Хвосты *.xlsx`), целевой KPI и ограничения.
Выход — ранжированные карточки проверяемых гипотез с механизмом влияния,
обоснованием с цитатами, оценкой эффекта на KPI (тонны Ni/Cu), рисками и
дорожной картой проверки; интерактивный дашборд с графом знаний,
корректируемым экспертом.

[source,bash]
----
make up        # весь стек в Docker: дашборд :8088, API :8000/docs
make test      # 36 тестов
----

== Ключевые свойства

* *Интерпретируемость* — каждое число в карточке трассируется до ячейки
  исходного отчёта; допущения выписаны явно.
* *Сменная нейронка* — LLM изолирована в отдельном сервисе за общим
  интерфейсом провайдера; модель меняется переменной окружения.
* *Оффлайн-режим* — при `LLM_PROVIDER=none` конвейер полностью работоспособен
  на rule-based движке (требование локального контура).
* *Эксперт в контуре* — свои гипотезы, правки графа, веса ранжирования,
  подтверждение/отклонение, 👍/👎 — всё мгновенно переранжирует выдачу.
